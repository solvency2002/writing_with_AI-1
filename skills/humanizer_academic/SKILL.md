# Academic Humanizer Demo

このファイルは、文体調整Skillの使い方を見せるためのローカル教材。
元の発想は `humanizer_academic` を参考にするが、このWS用に短くしたデモ版。

## 方針

- 学術的な正確性を優先する
- 直訳調を避け、自然な日本語にする
- 断定を強くしすぎない
- 不要な修飾語を減らす
- 1文を長くしすぎない
- 「重要である」「示唆される」などの定型表現を使いすぎない
- 研究デザイン、対象、アウトカム、限界が曖昧にならないようにする

## AIへの依頼時の注意

- 本文全体を一度に書き換えない
- 修正対象のセクションを指定する
- 変更理由を短く説明させる
- 引用は `refs.bib` にあるものだけを使わせる
